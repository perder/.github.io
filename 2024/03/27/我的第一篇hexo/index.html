npm 镜像库

在国内大家平常使用 npm 的时候可能会因为网络波动等原因，导致 install 的时候非常慢，这时候就需要配置镜像源，这样能够极大的提升下载速度。

国内镜像源

淘宝镜像:https://registry.npm.taobao.org（似乎已经过期）
**阿里云镜像:**https://npm.aliyun.com
**腾讯云镜像:**https://mirrors.cloud.tencent.com/npm/
**华为云镜像:**https://mirrors.huaweicloud.com/repository/npm/
**网易镜像:**https://mirrors.163.com/npm/
**中科院大学镜像:**http://mirrors.ustc.edu.cn/
**清华大学镜像:**https://mirrors.tuna.tsinghua.edu.cn/
